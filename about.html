About Cambridge Dashboard
Personal operations dashboard for commute, food, sports, EV charging, weather, and local activities.

What It Shows

  • CommuteLive Alewife Red Line departures.
  • LifestyleAlewife Bistro daily menu and weekend visual state.
  • SportsBoston pro-team games, status, logos, and venues.
  • EVGarage B ChargePoint port availability and notify toggle.
  • WeatherCurrent weather and adaptive visual scene.
  • CityBoston/Cambridge event feeds with de-duplication and filtering.

Refresh Behavior

  • Every 30s: MBTA departures, ChargePoint station status.
  • Every 60s: Bluebikes station status.
  • Every 10m: Weather data refresh.
  • Every 30m: Sports + local activities refresh.
  • Weekend layout auto-adjusts panel order and activity columns.

Data Sources

Menu JSON: GitHub raw menu
Weather: OpenWeather Current API
MBTA schedules: MBTA v3 API
Bluebikes station status: GBFS feed
ChargePoint station endpoint: ChargePoint map API endpoint
Sports scoreboards: ESPN site API
Events feeds: Boston.gov RSS and CambridgeUSA RSS
